Instalasi Server Local (Offline)
- Download dan instal XAMPP v3.2.1 atau dengan PHP 5.6
- Jalankan XAMPP untuk Apache dan MySQL

Instalasi
- Pindahkan folder SPK-AHP-ELECTRE ke folder htdocs pada XAMPP (misal C:\xampp\htdocs)

Buat Database
- Buka browser dan ketikkan localhost/phpmyadmin
- Buat database dengan nama spk_ahp_electre_ci
- Kemudian pilih tab import
- Import file yang ada pada folder Database

Jalankan Program Aplikasi
- Buka browser dan ketikkan localhost/SPK-AHP-ELECTRE
